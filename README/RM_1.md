## 📝 R3F란?

> `three.js` 를 기반으로 하는 `react` 라이브러리

`three.js`는 `WebGL`/ `WebGPU`를 `wrapping`한 라이브러리이다

`three.js`를 react에서 사용하기 위한 라이브러리가 `R3f` 이다
